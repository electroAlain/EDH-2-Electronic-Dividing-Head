# EDH-2 Electronic Dividing Head

## Project Overview
The **EDH-2 Electronic Dividing Head** is a precision tool designed for use in machining and woodworking. It allows users to efficiently divide angles and produce highly accurate workpieces through electronic control functionalities.

## Features
- **Automated Angle Division**: Achieves precise angle divisions with a simple user interface.
- **User-Friendly Interface**: Intuitive buttons and display for easy operation.
- **Versatile Hardware Compatibility**: Can be connected to multiple types of machines.
- **High Precision**: Ensures accuracy in machining processes.

## Hardware Connections
- **Power Supply**: Connect the device to a suitable power source (specified voltage).
- **Motor Connections**: Attach the stepper motor to the designated motor drive pins.
- **Control Interface**: Ensure that control interfaces are linked to the microcontroller.
- **Display Module**: Connect the display module for visual feedback on operations.

## Operation Guide
1. **Setup**: Ensure all hardware connections are secure before powering on the device.
2. **Power On**: Activate the device using the main power switch.
3. **Select Function**: Use the buttons to choose the desired function, such as angle division or manual control.
4. **Input Parameters**: Enter the required parameters using the interface.
5. **Execute**: Confirm the settings and execute the operation.

## Button Reference
- **Start/Stop**: Initiates or halts the current operation.
- **Adjust Angle**: Increases or decreases the angle setting.
- **Select Mode**: Cycles through available operational modes.
- **Reset**: Resets the current settings to default values.

## Calculation Examples
- **Dividing 90 Degrees into 5 Parts**: 
  - Enter 90 degrees on the control interface. 
  - Select "Divide" function. 
  - Result: Each division will be 18 degrees.

- **Calculating Gear Ratios**: 
  - For a 24-tooth gear driving a 48-tooth gear, 
  - Ratio = 48/24 = 2:1  

## Troubleshooting
- **Problem: Device Does Not Power On**
  - **Solution**: Check power supply connections and ensure the outlet is functioning.

- **Problem: Inaccurate Angle Division**
  - **Solution**: Recalibrate the device following the user manual instructions.

- **Problem: Unresponsive Interface**
  - **Solution**: Restart the device and check all connections.

For additional help, refer to the complete user manual or contact the support team.
